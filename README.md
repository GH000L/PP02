# PP02 — Производственная практика (Системное администрирование)

## Описание

Данный репозиторий содержит материалы производственной практики ПП.02 по направлению «Системное администрирование». В ходе практики выполнялись задачи по администрированию информационной системы организации ООО «Авторы Брокеридж», автоматизации работы системного администратора и обеспечению стабильной и безопасной работы компьютерной инфраструктуры.

Практика выполнялась в среде Windows с использованием PowerShell и средств администрирования операционной системы.

---

## Содержание репозитория

### docs/
Документация по практике:
- Отчёт по производственной практике
- Описание предметной области
- Техническое задание
- UML-диаграммы (Use Case, Class, Sequence, Activity, State, Deployment)

### scripts/
Скрипты администрирования PowerShell:
- `backup.ps1` — резервное копирование данных  
- `restore.ps1` — восстановление данных  
- `create_user.ps1` — создание пользователя  
- `disable_user.ps1` — отключение пользователя  
- `monitoring.ps1` — мониторинг состояния системы  
- `disk_check.ps1` — проверка состояния диска  
- `network_check.ps1` — проверка сети  
- `update_windows.ps1` — обновление системы  
- `cleanup.ps1` — очистка временных файлов  
- `antivirus_check.ps1` — проверка антивирусной защиты  
- `logs.ps1` — просмотр системных журналов  

---

## Цель работы

Целью производственной практики являлось получение практических навыков системного администрирования, автоматизация задач обслуживания системы, обеспечение безопасности данных и повышение надежности работы информационной инфраструктуры.
